# Plugin: Student Manager (Quản lý Sinh viên)

### 1. Giới thiệu
Plugin cho phép quản trị viên quản lý danh sách sinh viên với các thông tin: MSSV, Họ tên, Chuyên ngành, Ngày sinh.

### 2. Chức năng chính
- **Backend:** Đăng ký Custom Post Type "Sinh viên", thêm Meta Box nhập liệu an toàn.
- **Frontend:** Hiển thị danh sách sinh viên dưới dạng bảng qua Shortcode `[danh_sach_sinh_vien]`.

### 4. Cách cài đặt
1. Tải file `.zip` lên mục Plugin của WordPress.
2. Kích hoạt Plugin.
3. Thêm sinh viên mới.
4. Tạo trang và chèn Shortcode `[danh_sach_sinh_vien]`.
